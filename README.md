**Interactive Resume Builder**
A minimalist, browser-based tool to create a simple resume.
Features
**Direct Editing**: Click any text to edit instantly.
**PDF Export**: Save a print-ready A4 document via html2pdf.js.
**No Install**: Runs entirely in the browser without a backend.

Open the file:
Launch index.html in any web browser.
Create:
Click placeholder text to enter your details.
Export:
Click "Download as PDF" to save your file.


**Tech Stack**
HTML5, CSS3, Vanilla JavaScript.
Library: html2pdf.js (via CDN).
